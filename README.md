<h1 align="center">爱摸鱼主站 👋</h1>


> 使用 Vite + Vue3 + AntDesignVue + Pinia 搭建的在线摸鱼网站

## 运行、打包

- #### 安装

  ```
  npm install
  ```

- #### 运行

  ```
  npm run serve
  ```

- #### 打包

  打包前请注意将`.env.production`的`VITE_BASE_PATH`修改为`/`

  ```
  npm run build
  ```

## 技术栈

- Vite
- Vue3
- Ant Design Vue
- Pinia
- file-saver
- mitt
- pinia-plugin-persist
- vue3-perfect-scrollbar
- vuedraggable
- less
